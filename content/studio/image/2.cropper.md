# Image Cropper

The cropping tool allows users to select a specific rectangular or circular area of an image. It supports fixed aspect ratios, grid line overlays, and constrained resizing.

The standard cropper with default settings (80% initial crop area, resizable, and movable).

## 04. Social Media Banner (16:9)

A wider viewport for hero images or banners. This example forces a **16:9 aspect ratio** and sets a large **initial crop area**.

::studio-view{name="cropper"}
---
src: https://images.pexels.com/photos/3183153/pexels-photo-3183153.jpeg
props:
  cropper:
    aspectRatio: 1.7777777778
    initialCropPercent: 95
---
::

## 05. Fixed Window (Zoom & Pan)

A high-end "fixed-window" experience where the crop area is locked in the center, and the user **zooms and pans the image** behind it.

::studio-view{name="cropper"}
---
src: https://images.pexels.com/photos/4323307/pexels-photo-4323307.jpeg
props:
  stencil:
    fixed: true
    aspectRatio: 1
  cropper:
    initialCropPercent: 50
---
::

## Advanced Options Reference

| Property | Type | Default | Description |
| --- | --- | --- | --- |
| `aspectRatio` | `number` | `undefined` | The ratio of width to height for the crop area. |
| `initialCropPercent` | `number` | `80` | The initial size of the crop area (0-100% of the image size). |
| `gridLines` | `boolean` | `true` | Show or hide the rule-of-thirds grid. |
| `stencil.type` | `'rect' \| 'circle'` | `'rect'` | The shape of the cropping mask. |
| `stencil.fixed` | `boolean` | `false` | If true, the image is zoomed/panned while the stencil remains fixed. |

---

::tip
Combine **stencil.fixed: true** with **stencil.type: circle** for a modern, WhatsApp-like avatar cropping experience.
::
